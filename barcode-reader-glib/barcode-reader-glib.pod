=head1 NAME

barcode-reader - receive barcodes from the B<me.koppi.BarcodeReader> service. 

=head1 SYNOPSIS

barcode-reader [options...]

=head1 DESCRIPTION

B<barcode-reader> receives barcodes from the DBus B<me.koppi.BarcodeReader> service and prints them to the standard output in the format read 'basrcode' newline.

=head1 OPTIONS

=over 8

=item B<--help>

Prints a help message and exits.

=item B<--verbose>

Be more verbose about the things going on in the background.

=item B<--version>

Displays information about the currently installed version and exists.

=back
